# SÓCRATES

**Sistema de Orientação Cognitiva, Raciocínio Ativo, Tutoria Educacional Soberana**

> A IA que se recusa a pensar por você.

---

## O que é

SÓCRATES é um agente educacional que opera sob um princípio radical: **nunca dar a resposta pronta**.

Inspirado no método socrático, o agente constrói conhecimento junto com o estudante através de perguntas, provocações e diálogo — nunca por transmissão passiva. Para mediadores (professores), oferece apoio incondicional com Design Universal para Aprendizagem (DUA) integrado de forma invisível.

## Pilares

### 1. Soberania Humana
O controle cognitivo permanece com o ser humano. A IA é ferramenta, nunca oráculo. O estudante *studere* (busca ativamente) — nunca *alere* (recebe passivamente).

### 2. Maiêutica
O motor do sistema é socrático: perguntas que forçam o estudante a pensar, conectar, analisar, criar. Máximo 2 frases de conteúdo factual por turno. Depois: provocação cognitiva obrigatória.

### 3. Design Universal para Aprendizagem (DUA)
Múltiplos meios de representação, engajamento e expressão. Opera invisível — o estudante nunca sabe que está sendo adaptado. Cada cérebro tem sua arquitetura.

### 4. Ética Arendtiana
Inspirado no conceito de banalidade do mal de Hannah Arendt: o perigo não é a maldade, é a ausência de pensamento. Quando detecta padrões de não-pensamento — reprodução acrítica, adesão automática, recusa de examinar — o agente usa maiêutica para provocar reflexão. Nunca julga. Nunca apaga a história. Nunca proíbe. Apenas faz pensar.

## Para Estudantes

- Parceiro socrático: constrói o caminho até a resposta, nunca entrega pronta
- Adaptação silenciosa ao perfil cognitivo de cada estudante
- Fontes exclusivamente acadêmicas, mediadas por nível
- Sempre oferece um degrau além do currículo mínimo (BNCC)
- Respeita a arquitetura cognitiva de cada cérebro — neurodivergente ou não

## Para Mediadores(as)

- Apoio incondicional ao planejamento pedagógico
- DUA aplicado automaticamente em toda sugestão
- Barreira rebaixada: linguagem direta, aplicação imediata, sem jargão
- Relatório de observação pedagógica com linguagem protegida
- Modo *Chronos* (logística direta) e modo *Kairos* (aprofundamento maiêutico)
- Alertas respeitosos sobre tendências pessoais ou doutrinárias no planejamento

## Princípios

1. Ninguém aprende recebendo respostas
2. O conforto excessivo atrofia
3. Cada cérebro tem sua arquitetura
4. O mínimo curricular não é o máximo
5. O mediador é aliado, não alvo
6. Tecnologia sem soberania é anestesia

## Como testar

### Opção 1 — Claude (Anthropic)
1. Abra [claude.ai](https://claude.ai)
2. Crie um novo Projeto
3. Solicite acesso à constituição via [Issues](../../issues) deste repositório
4. Cole nas instruções do projeto e inicie uma conversa

### Opção 2 — ChatGPT (OpenAI)
1. Abra [chatgpt.com](https://chatgpt.com)
2. Vá em "Criar um GPT" (precisa de Plus)
3. Solicite acesso à constituição e cole nas instruções
4. Publique como link privado

### Opção 3 — Qualquer LLM com system prompt
1. Solicite acesso à constituição
2. Use como system prompt na API ou interface do LLM de sua escolha

> **Nota:** A constituição (system prompt) não é publicada neste repositório para proteger a integridade do método. Educadores interessados em testar podem solicitar acesso abrindo uma Issue.
